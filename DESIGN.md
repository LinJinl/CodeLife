# 《道途·学海》—— 修仙风格个人学习网站 · 设计文档 v1.0

---

## 一、核心概念

### 世界观设定

> 你不是在学习，你是在**修仙**。
> 每一篇博文是一次「顿悟」，每一道算法题是「炼丹」，每一次 GitHub 提交是「铸剑」。
> 日积月累，终成仙道。

网站以中国修仙小说的境界体系为骨架，将日常的学习行为映射为修炼行为。
用户的学习数据驱动着角色的境界成长，把「枯燥的坚持」变成「史诗的修炼」。

### 网站命名

**主名**：`道途` （The Dao Path）
**副标题**：`一个修仙者的藏经阁`
**域名方向**：`daotu.dev` / `xianjing.me` / `linjl.dev`

## 二、境界体系（等级系统）

### 修仙境界 × 学习积累

| 境界 | 所需修为（累计积分） | 对应学习量（大致） |
|------|------|------|
| 炼气期 · 一重 | 0 | 刚刚开始 |
| 炼气期 · 九重 | 500 | 写了几篇文章 |
| 筑基期 | 1,500 | 形成学习习惯 |
| 金丹期 | 5,000 | 持续输出数月 |
| 元婴期 | 15,000 | 深度积累 |
| 化神期 | 40,000 | 领域专家级别 |
| 炼虚期 | 100,000 | 传道授业 |
| 合体期 | 250,000 | 开宗立派 |
| 大乘期 | 600,000 | 宗师 |
| 渡劫 · 飞升 | 1,000,000+ | 传说 |

### 修为获取规则

| 行为 | 修为获取 | 称呼 |
|------|------|------|
| 发布博文（>500字） | +80 修为 | 「顿悟」|
| 发布博文（>2000字） | +200 修为 | 「大悟」|
| 解出 LeetCode Easy | +30 修为 | 「小试炼」|
| 解出 LeetCode Medium | +80 修为 | 「中试炼」|
| 解出 LeetCode Hard | +200 修为 | 「大试炼」|
| GitHub 每次 Commit | +15 修为 | 「铸剑一锤」|
| 连续打卡（7天） | +500 修为 | 「七日不辍」|
| 连续打卡（30天） | +3000 修为 | 「月修不断」|
| 读完一本书（记录） | +300 修为 | 「览典」|

### 特殊成就（仙缘）

- **「天才之名」**：单日修为突破 500
- **「道心坚固」**：连续30天不间断
- **「算法宗师」**：累计解题 500 道
- **「著书立说」**：发布 100 篇博文
- **「万码长城」**：累计 commit 超过 1000 次

## 三、视觉设计语言

### 3.1 风格定义

**核心美学**：`水墨·赛博·修仙`

融合：
- **水墨画**的晕染质感、留白美学
- **赛博朋克**的霓虹光效、科技感线条
- **古风**的汉字构图、卷轴感排版

**关键词**：幽深 / 灵气流动 / 古朴与科技并存 / 史诗感

### 3.2 色彩系统

```
主色调（深空黑）   #0A0B0F  ← 背景底色，极深的蓝黑
次级背景           #0F1118  ← 卡片/面板背景
表面层             #161922  ← 内容区域背景

灵气青（主题色）   #00E5FF  ← 修为、进度、高亮
仙光金（点缀色）   #FFD700  ← 境界标题、成就
渡劫紫（特殊色）   #8B5CF6  ← 稀有成就、最高境界
丹炉红（警示色）   #FF4444  ← 危险、失败、警告
生机绿（成功色）   #00FF88  ← 完成、成功、通关

文字主色           #E8E8F0  ← 主要文字
文字次色           #8890A0  ← 辅助说明文字
文字暗色           #4A5060  ← 禁用/占位文字

边框色             rgba(0, 229, 255, 0.15)  ← 灵气边框
光晕色             rgba(0, 229, 255, 0.05)  ← 背景微光
```

### 3.3 字体系统

```
标题字体：「ZCOOL XiaoWei」（站酷小薇体）—— 古典优雅，带毛笔感
副标题：「Noto Serif SC」（思源宋体）—— 正统汉字，用于境界名称
正文：「ZCOOL QingKe HuangYou」/ 「Noto Sans SC」—— 可读性高
数字/代码：「JetBrains Mono」—— 程序员美学，等宽字体
英文点缀：「Cinzel」—— 史诗感衬线字体，用于英文标题
```

### 3.4 核心视觉元素

**粒子系统**：页面背景漂浮着若隐若现的灵气粒子（细小光点），缓慢游动
**扫光效果**：境界提升时，从屏幕底部涌起金色光浪
**水墨晕染**：卡片hover时，边缘出现水墨扩散效果
**符文线条**：装饰性的抽象汉字笔画，半透明叠加在背景
**进度灵脉**：进度条用流光效果代替普通填充，像活的灵脉

## 六、交互动效设计

### 6.1 境界提升动画（最重磅时刻）

触发条件：修为积累到达下一境界阈值

动画流程：
1. 屏幕四角出现灵气粒子向中心汇聚（0.5s）
2. 背景光晕爆发，整个页面短暂变亮（0.3s）
3. 顶部出现大字横幅：「恭喜道友，突破 [境界名] !」（宋体，金色，从中心放大出现）
4. 配合音效（可选）：清越的剑鸣声
5. 动画缓慢消退（1s），页面恢复，导航栏中的境界标签更新

### 6.2 修为获取微动画

每次记录新行为后：
- 修为数字滚动更新（数字翻滚动画）
- 进度条以「流水」动效向右推进
- 右上角短暂弹出：「+80 修为 · 炼丹成功 ⚗️」

### 6.3 页面切换

- 使用「卷轴展开」效果：从上到下展开内容
- 或「墨迹晕染」：内容区域从中心向外扩散显现

### 6.4 Hover 效果

- 卡片：边框微微发光，有一层蓝色光晕在边缘流动
- 按钮：轻微上浮（translateY -2px），阴影加深
- 链接：下划线用流光扫过效果，而非普通下划线
